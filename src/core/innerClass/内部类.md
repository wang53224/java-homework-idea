# 内部类

> 内部类是定义在另一个类中的类

## 为什么需要内部类？

1. 内部类方法可以访问该类定义所在的作用域中的数据，包括私有的数据。
2. 内部类可以对同一个包中的其他类隐藏起来。
3. 定义一个回调函数且不想写大量代码时，使用匿名内部类比较便捷。
